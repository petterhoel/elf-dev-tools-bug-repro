# ELF POC

Small in memory .Net 6 web api and Angular 13 app with Elf

You will ned [.Net 6 sdk](https://dotnet.microsoft.com/download) installed and npm/yarn

## Folders
```
<repo>
  |-api/
  |-client/
```

## Run webapi (.Net 6)
```
cd api
dotnet watch
```

## Install npm deps
```
cd client
npm i
```

## Run front end app
```
cd client
npm start
```
