# ELF POC

_!!!dette er en WIP!!!_

Bittelite in memory web api og en webapp som bruker elf for state management

## Mappestruktur
```
<repo>
  |-api/
  |-client/
```

## Kjør apiet (.Net 6)
```
cd api
dotnet watch
```

## Installer npm pakker
```
cd client
npm i
```

## Kjør frontend
```
cd client
npm start
```
